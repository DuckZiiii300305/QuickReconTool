# KRX - Kali Recon X

KRX is a CLI-based automated reconnaissance tool designed for Kali Linux.

## Features

- Asset Enumeration (Amass)
- Live Host Detection (httpx)
- Port Scanning (Nmap)
- Directory Brute-force (ffuf)
- Vulnerability Scan (Nuclei)
- HTML Report Generation

## Installation

```bash
git clone https://github.com/yourname/krx.git
cd krx
chmod +x install.sh
./install.sh
